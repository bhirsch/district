District
_________

To install place the entire District folder into your modules directory.

Go to Administer -> Site configuration -> District, save your Sunlight Labs API key and then press the buttons below to load district data, create taxonomy terms and apply terms to related nodes.

If you do not have a key yet, you can get one here: http://services.sunlightlabs.com/api/register/

Once the module is installed, update locations with District tags by going to mysite.com/admin/settings/district and clicking "Update Locations Without District Tags."



Dependencies 
_____________
This module depends on the Location module.
